Setup Hardat

Hardat is a framework that will help us to :

  • compile our smart contract
  • test our smart contract
  • deploy our smart contract

If you feel adventurous you can alternatively use Truffle or Fundry

Install hardat npm install --save-dev hardhat

Then init hardat with npx hardhat
